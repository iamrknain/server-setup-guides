## Server Setup Guides

This repository contains practical documentation and guides for configuring and managing Linux-based servers. The notes focus on common tasks required for deploying and maintaining web services, development environments, and monitoring tools.

Topics covered include:

* Apache HTTP server configuration
* HTTP and HTTPS setup for web services
* Matomo analytics server deployment
* MySQL database management
* Python virtual environment setup
* Linux virtual machine administration
* General server management practices

The repository is intended as a reference for developers and system administrators when i needed quick documentation for common server configuration tasks and deployment workflows.
